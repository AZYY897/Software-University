is a style description language (style sheet language) - used primarily to describe the presentation of a document written in markup language. Most commonly used in conjunction with HTML , but can be applied to any XML document. The CSS specification is officially supported by W3C . CSS was created to separate the content and structure of web pages separately from their visual presentation. Prior to the CSS standards established by W3C in 1995, site content and design style were written in the same HTML page. As a result, HTML code becomes complex and illegible, and any change in the design of a site requires that the correction be applied throughout the site page by page. Using CSS, the formatting settings can be put into a single file, and then the change will be reflected simultaneously on all pages that use that CSS file.
